There are two files from Stack Overflow:
1. survey_results_public.csv - CSV file with main survey results, one respondent per row and one column per answer
2. survey_results_schema.csv - CSV file with survey schema, i.e., the questions that correspond to each column name
The enclosed data set is the complete, cleaned results of the 2021 Stack Overflow Developer Survey. 

There are several files generated by Joe:
1. survey_demographics.csv -CSV file with only demographics information, deducted version of survey_results_public.csv
2. survey_dev.csv -CSV file with developer's job title, coding year and salary(yearly in the US dollars), one respondent per row but one respondent may have many rows due to their different titles
3. survey_techs.csv -CSV file with developer's working and desiring technologies, one respondent per row but one respondent may have many rows due to their many picks on technologies
4. wrangling_developer_type.ipynb -Implement data wrangling with Python
5. wrangling_techs.ipynb -Implement data wrangling with Python
